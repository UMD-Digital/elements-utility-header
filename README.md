# &lt;umd-utility-header&gt; element

## Installation

```
$ yarn add @umd/utilityheader
```

## Usage

```js
import '@umd/utilityheader';
```

```html
<umd-utility-header
  search
  events
  news
  schools
  admissions
  gift
  size="full"
  padding="20"
>
</umd-utility-header>
```
